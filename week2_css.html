CSS Box Model & Flexbox

Understanding layout fundamentals

CSS Box Model

Box Model Properties

Content: The actual content area
Padding: Space inside the border
Border: The border around padding
Margin: Space outside the border

CSS Flexbox

Flexbox Properties:

display: flex
justify-content
align-items
flex-direction

Flexbox Example

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 20px;
}

.item {
    flex: 1;
    padding: 20px;
}
