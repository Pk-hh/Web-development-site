DOM Manipulation & Events

Making web pages interactive and responsive

DOM (Document Object Model)

Select elements: getElementById()
Change content: textContent
Modify styles: style.property

Common Events

click
submit
mouseover
keydown

Interactive Example 

// HTML
<button id="colorBtn">Change Color</button>
<div id="box">Click the button!</div>

// JavaScript
const btn = document.getElementById('colorBtn');
const box = document.getElementById('box');

btn.addEventListener('click', () => {
    box.style.backgroundColor = '#ff6b6b';
    box.textContent = 'Color changed!';
});
